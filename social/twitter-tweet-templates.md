## 4. Educational / Value (4 tweets)

### Tweet 4.1
```
Here's why IV Rank matters more than IV itself:

IV = 30% could be LOW for TSLA and HIGH for JNJ.

IV Rank tells you where 30% sits *relative* to the past year.

>70% = expensive → sell premium
<30% = cheap → buy premium

Context > absolute value. Always.

#OptionsEducation #Volatility
```

### Tweet 4.2
```
The 3 setups I trade 80% of the time:

1. VWAP bounce (CALL) — trend day, pullbacks hold
2. ORB reject (PUT) — first 5min fail, fade to VWAP
3. Gap fade (PUT) — overextended gap, mean reversion

No need for 20 setups. Master 3. Execute with size.

#DayTrading #VWAP
```

### Tweet 4.3
```
0DTE vs Weekly options:

0DTE: Gamma risk, tighter stops, need conviction
Weekly: More room, wider stops, can let thesis play out

I use 0DTE when VIX < 20 (calm = directional works).
I use weeklies when VIX > 25 (chop = time needed).

VIX is your regime filter. Use it.

#0DTE #OptionsStrategy
```

### Tweet 4.4
```
The biggest mistake I see in options trading:

Buying calls because "the stock is going up."

No. You buy calls when:
✅ Direction is clear
✅ IV Rank is low (<30)
✅ Risk/reward is ≥ 2:1
✅ You have a stop

Direction alone is not enough.

#OptionsTrading #RiskManagement
```
