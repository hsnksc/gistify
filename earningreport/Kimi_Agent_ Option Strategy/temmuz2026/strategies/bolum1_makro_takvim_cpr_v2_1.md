## 4. CALL/PUT RATIO (CPR) REHBERİ

### 4.1 CPR Nedir ve Nasıl Yorumlanır?

**Call/Put Ratio (CPR)**, belirli bir hissenin opsiyon piyasasında call işlem hacminin put işlem hacmine oranını gösterir. EarningsPlay v4 metodolojisinde temel bir göstergedir.

| CPR Aralığı | Sentiment | Ne Anlama Gelir? | Strateji Eğilimi |
|-------------|-----------|-----------------|------------------|
| **< 0.60** | Güçlü Ayı | Put hacmi call'ın 1.67x üzerinde. Piyasa korkulu, put'lar şişmiş | Bull Put Spread, Long Call |
| **0.60 - 0.80** | Ayı-Eğilimli | Daha çok put işlem var. Korunma talebi yüksek | Bull Put Spread, Asimetrik IC |
| **0.80 - 1.00** | Nötr-Acı | Dengeli ama hafif ayı baskısı | Simetrik Iron Condor |
| **1.00 - 1.25** | Nötr-Boğa | Dengeli ama hafif boğa baskısı | Simetrik IC, Bull Call Spread |
| **1.25 - 2.00** | Boğa-Eğilimli | Daha çok call işlem var. Yükseliş beklentisi | Bear Call Spread, Asimetrik IC (Put ağır) |
| **> 2.00** | Güçlü Boğa | Call hacmi put'un 2x üzerinde. Aşırı iyimserlik | Bear Call Spread, Short Call |

### 4.2 Şişmiş Opsiyon ve Strateji Eşleştirme

| Hacim CPR | Şişmiş Opsiyon | Eşleşen Strateji | Call/Put Kısa Oranı (IC) |
|-----------|---------------|-------------------|---------------------|
| **< 0.60** | Put'lar AŞIRI şişmiş | Bull Put Spread / Long Call | %70 Call / %30 Put |
| **0.60 - 0.75** | Put'lar şişmiş | Bull Put Spread / Asimetrik IC | %60 Call / %40 Put |
| **0.75 - 1.00** | Nötr (Dengeli) | Simetrik Iron Condor | %50 Call / %50 Put |
| **1.00 - 1.50** | Call'lar hafif şişmiş | Asimetrik IC (Put ağır) | %40 Call / %60 Put |
| **> 1.50** | Call'lar şişmiş | Bear Call Spread / Short Call | %30 Call / %70 Put |

### 4.4 EarningsPlay v4 Metodolojisi Özeti — EARNINGS PLAY VERSİYONU

| Parametre | Kural |
|-----------|-------|
| **IV Rank > %50** | Iron Condor (Ana Strateji) |
| **IV Rank %40-50** | Long Spread veya Marjinal IC |
| **IV Rank <%40** | Long Spread, Butterfly, Calendar |
| **Wing Width** | Hisse fiyatı / 10 |
| **Short Strikes** | Expected Move'ün %10-15 dışında |
| **Entry** | Earnings'ten **2-5 gün önce** |
| **Exit** | Earnings sonrası **1-2 gün içinde** (IV crush sonrası) |
| **Max Hold** | **2 gün** (⛔ earnings play'ler uzun vadeli değildir) |
| **Kar Hedefi** | Toplanan kredinin/primin **%50-75'i** (IV crush hızlı kazanç) |
| **Max Kayıp (Stop)** | Toplanan kredinin 2.0x'i (%200) |
| **Delta Toleransı** | ±0.10 |
| **Pozisyon Boyutu** | Hesabın %1-2'si (VIX>25'da %1, VIX>30'da %0.5) |
| **VIX < 15** | Normal pozisyon, geniş IC |
| **VIX 15-25** | Normal pozisyon, wider wings |
| **VIX 25-35** | %50 azalt, dar IC veya Long Straddle |
| **VIX > 35** | %75 azalt veya nakit |

> **⛔ ESKİ KURALLAR KALDIRILDI:** "21 DTE çıkış", "Zamanla çıkış", "Ağustos'ta çıkış" gibi ifadeler Earnings Play formatında YOKTUR. Tüm earnings play'ler earnings sonrası 1-2 gün içinde kapatılır.
